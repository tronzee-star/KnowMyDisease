# KnowMyDisease
KnowMyDisease is a user-friendly web application that allows users to search for common diseases and instantly view detailed information including descriptions, symptoms, causes, and recommended medications. Designed with simplicity and clarity in mind,KnowMyDisease provides an accessible tool for students and patients.

## Features

- Search functionality to look up diseases.
- Displays disease description, symptoms, causes, and medication.
- Navigation bar with links to Home, List of Diseases, and About pages.
- Caution note to advise users to seek professional medical advice if needed.
- Clean, responsive design using HTML, CSS, and JavaScript.


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Example Diseases Included

- Asthma
